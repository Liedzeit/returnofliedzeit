In meiner Jugend las ich gern und viel Comics. Aber noch lieber Sekundärliteratur über Comics (und Trivialliteratur im Allgemeinen). Und eines der ersten Bücher war *Zok roarr wumm* von Karl Riha. Der starb nun und wird in der FAZ von Andreas Platthaus gewürdigt.

Da stellt sich heraus, dass der Mann auch ein begnadeter Dichter war. Abgedruckt wurde ein wunderschönes Sonett, das so losgeht:

<ul class="no-bullets">
<li>so weit so gut so zart so nett</li>
<li>so viel so kurz so stein so brett</li>
<li>so voll so leer so schwarz so breit</li>
<li>so still so klar so fluß so zeit</li>
</ul>
